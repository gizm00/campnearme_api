State of the Data

What exactly are we dealing with here?

This API provides access to the backend database for campnear.me
  • Includes 3,000+ campgrounds in Oregon and Washington
  • Consolidation of info from Federal, state and local agencies
  • Reservation availability for the upcoming weekend
  • Facility URLs from a variety of sources pertaining to each facility

Some things to keep in mind:
  • Latitude/Longitude coordinates for the campgrounds are mostly not verified. Lost Lake campground is, for example, not actually in the middle of a lake as the Lat/Long would have you believe
  • There are duplicates in the dataset
  • There are URLs that end in 404s in the dataset
  • There will be times when this data will (gasp) be wrong.

The intent of this dataset is to consolidate available sources of campground information to give campers a better springboard for trip planning. It is certainly necessary to take information with a grain of salt and call the local ranger station before heading into an unfamiliar part of the woods
