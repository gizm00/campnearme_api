Toorcamp!

Greetings Toorcampers!

I'm delighted to be hosting the first campnear.me API hackathon at Toorcamp! Hacking on campgrounds while camping, I cant think of a better situation to launch the campnear.me API

I'm excited to see what people make with the data, and wanted to provide some APIs that play nicely with campground data.
  • Forecast.io I use Forecast.io on campnear.me to provide forecast info for trip planning. They have a great drop in widget that you can just plop into your website with the corresponding lat/longs.

  • OpenSignal API Find out if you can tether and work from camp, or just see if you will have cell service at the campground, or search for specifically off the grid campgrounds :)

  • Inciweb, OR An RSS feed is available, provides latest information on forest fires. Definitely something I wish I'd had access to last summer camping near Lake Chelan!

  • ClearDarkSky For star gazers and astrophotogs, check out these forecast maps you can embed in your website, so long as you arent charging for access to it

The campnear.me API is extremely new, should you encounter issues please let me know on the GitHub page

New to data hacking? You can get a setup and some instructions from my tutorial on data pipelining in Python
Lab 1 should get you setup to get connected to an API endpoint and read the json response into a table-based object called a DataFrame. To learn more about python dataframes I recommend this tutorial from Julia Evans. You can run it right in the browser!

Since we will be camping, and this API is newly minted, I will have some USB sticks with a dump of the database should folks run into trouble accessing the API

If you build something and want to share, please consider using the toorcampnearme GitHub org! It would be swell to catalog what people make. Ping me with your GitHub handle to get added to the org.
