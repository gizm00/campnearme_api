campnear.me API

API Examples

Presently there are 3 endpoints for the campnear.me API. It is poorly written, likely buggy, and probably slow if more than a few people are pinging it. It will get better, I promise! When you do run into problems I would appreciate if you could let me know by filing an issue on the campnear.me API GitHub page. The API returns json, with an Items array, Status field, and NumRecords field

http://api.campnear.me/GetFacilityNames

This endpoint will return the list of all facility names in the campnear.me database. Wondering if some specific sites are catalogued? Heres one way to find out.
You can try it here!

http://api.campnear.me/GetAllFacilities

Contrary to its name, GetAllFacilities wont actually return all 3000x48 fields to you, since that seemed like a bad idea. Instead, GetAllFacilities without any additional parameters, will return the first 50 records (campgrounds) to you You can find out what that looks like here
http://api.campnear.me/GetAllFacilities.
Note the campnear_id field, this is a sequential field you can use to page through the data in subsequent queries

If you arent satisfied with 50 campgrounds and want moar, simply note the last campground_id in the result set and start right after that in another query:
http://api.campnear.me/GetAllFacilities?start_id=51

And finally, if you want less than 50 records you can modify the number of records retrieved with the 'limit' parameter:
http://api.campnear.me/GetAllFacilities?start_id=51&limit=2
Note: limit must be <= 50.

http://api.campnear.me/GetFacilitiesNear

The GetFacilitiesNear endpoint returns up to 50 records within the geographic area you define:
  • lat: latitude in degrees
  • lon: longitude in degrees
  • radius: search radius in degrees
For example, to search for campgrounds within a 20 mile radius of Portland OR you can perform the following query:
http://api.campnear.me/GetFacilitiesNear?lat=45.23&lon=-122.42&radius=20
You will see a field "D" in the resulting json which provides the exact distance (in miles) between the input lat/long and the campground.
The radius is limited to 100 miles.
