campnear.me API

API Examples

Presently there are 5 endpoints for the campnear.me API. It is poorly written, likely buggy, and probably slow if more than a few people are pinging it. It will get better, I promise! When you do run into problems I would appreciate if you could let me know by filing an issue on the campnear.me API GitHub page. The API returns json, with an Items array including the campground data and a NumRecords field with the total records returned. You can learn about the fields returned by the API on the data definition page

http://api.campnear.me/GetAllFacilities

Contrary to its name, GetAllFacilities wont actually return all 4000x100 fields to you, since that seemed like a bad idea. Instead, GetAllFacilities without any additional parameters, will return the first 50 records (campgrounds) to you. http://api.campnear.me/GetAllFacilities

If you arent satisfied with 50 campgrounds and want moar, simply note the last campground_id in the result set and start right after that in another query: http://api.campnear.me/GetAllFacilities?start_id=51

And finally, if you want less than 50 records you can modify the number of records retrieved with the 'limit' parameter: http://api.campnear.me/GetAllFacilities?start_id=51&limit=2 Note: limit must be <= 50.

http://api.campnear.me/GetFacilitiesNear

The GetFacilitiesNear endpoint returns facilities near a given lat long with a radius up to 50 miles. This endpoint returns the campnear_id that you can use with the GetFacilityDetails endpoint
  • lat: latitude in degrees
  • lon: longitude in degrees
  • radius: search radius in miles

For example, to search for campgrounds within a 20 mile radius of Portland OR you can perform the following query:
http://api.campnear.me/GetFacilitiesNear?lat=45.23&lon=-122.42&radius=20
You will see a field "distance" in the resulting json which provides the exact distance (in miles) between the input lat/long and the campground.

http://api.campnear.me/GetFacilityDetails

Retrieve the facility details for a given campnearme_id. http://api.campnear.me/GetFacilityDetails?campnear_id=1026.

http://api.campnear.me/GetAvailableFacilities

Retrieve all facilities (with full details) that have availability. http://api.campnear.me/GetAvailableFacilities

http://api.campnear.me/GetDistanceBetween

Get distance between two lat long points using the vincenty distance algorithm http://api.campnear.me/GetDistanceBetween?lat1=45.5173935&lon1=-122.8242758&lat2=47.6149938&lon2=-122.4763348
