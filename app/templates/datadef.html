Identifying Fields

"campnear_id": Int. Unique Id used to identify campgrounds. Use this for setting start_id in API queries
"facilityname": String. Campground name

Contact fields

"telephone": Campground telephone number
"recreationgov_url": String. If populated, gives the recreation.gov URL for the campground
"fs_link": Link to the usds.fs* site for the campground, if available
"site_url": URL for the campground, typically from the agency which runs the facility
"other_link": additional campground URL if available
"reserveamerica_url": URL for reserveamerica.com if available
"reservation_link": URL to query for reservation information

Location Fields

"latlong": Float,Float. Campground lat and long, i.e. 42.659147,-123.638272. NOTE the minus for longitude
"facilitylatitude": Float.
"facilitylongitude": Float.
"elevation": elevation of the campground in feet above sea level.
"driveminutes": Number of minutes to drive from Portland, OR - pair with drivehours below
"drivehours": Number of hours to drive from Portland, OR - pair with driveminutes above
"drivemiles": number of miles from Portland, OR to the campground
"transitminutes": Number of minutes on public transit from Portland, OR to campground. Pair with transithours below.
"transithours": Hours from Portland,OR to campground for public transit if available. Pair with transitminute above

Campground Amenity Fields

"cg_lookout": True/False. Indicates of campground entry is a lookout
"cg_conditions": String. Shows current conditions (as of last update)
"cg_reservable": True/False. Indicates of campground is reservable
"cg_usage": String. Typically "light","medium","heavy" to indicate how heavily used the campground is
"cg_group": True/False. Indicates if campground has group camping
"cg_solitude": True/False. Special sauce of determining if this is a good place for solitude or not
"cg_nearriver": True/False. Indicates if campground is near a river
"cg_fcfs": True/False. Indicates if campground is first come, first served
"cg_nearbeach": True/False. Indicates if the campground is on a beach. NOTE - does not guarantee that campSITES are on the beach.
"cg_transit_avail": True/False. Indicates if public transit is available from Portland, OR to this campground
"cg_proximitytowater": True/False. Indicates if the campground is near bodies of water - lakes, rivers, ocean
"cg_season": String indicating operation dates/seasons
"cg_status": String. Campground status as of the last db update
"cg_nearlake": True/False. Indicates if campground is near a lake
"cg_accessible": True/False. Indicates if the campground has accessible campsites
"cg_vault": True/False. Indicates if vault toilets are available
"cg_flush": True/False. Indicates if flushing toilets are available at the campground
"cg_shower": True/False. Indicates if showers are available at the campground
"cg_water": String - false/unknown/true/seasonal - indication of water availability onsite
"cg_petsallowed": True/False. Indicates if campground has an established pet policy
"nofee": String/ True/False. Indicates if campground has no fee for camping
"yurts_available": True/False. Indicates if campground has yurts - NOT an idicator of yurts being available to reserve
"cabins_available": True/False. Indicates if there are cabins available at the campground - NOT an idicator of cabins being available to reserve

Reservability Scraping Fields

"start_date": mm/dd/YYYY indicating the start day (Friday) of the reservation window
"end_date": mm/dd/YYYY indicating the end day (Sunday) of the reservation window
"last_updated": mm/dd/YYYY indicating the last time availability for the above window was checked

Reservability Site Availability Fields

"sites_available": Number of sites available for above reservation window
"total_sites": Number of total sites at the campground
