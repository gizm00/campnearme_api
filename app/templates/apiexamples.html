Here are some examples to show you how to use the campnear.me API

http://api.campnear.me/GetFacilities

optional parameters: start_id = facilityindex to start from, limit = number of records to return (max 50) Without any additional parameters, GetFacilities will return the first 50 records in the database: http://api.campnear.me/GetFacilities.

To get more campgrounds, simply note the last facilityindex in the result set and start right after that in another query:
http://api.campnear.me/GetFacilities?start_id=51

And finally, if you want less than 50 records you can modify the number of records retrieved with the 'limit' parameter:
http://api.campnear.me/GetFacilities?start_id=51&limit=2

http://api.campnear.me/GetFacilityDetails

Retrieve the facility details for a given facilityindex specified in the campnear_id argument. http://api.campnear.me/GetFacilityDetails?campnear_id=1026.

http://api.campnear.me/GetFacilitiesNear

The GetFacilitiesNear endpoint returns up to 50 records within the geographic area you define:
  • lat: latitude in degrees
  • lon: longitude in degrees
  • radius: search radius in miles (limit 50)
For example, to search for campgrounds within a 20 mile radius of Portland OR you can perform the following query:
http://api.campnear.me/GetFacilitiesNear?lat=45.23&lon=-122.42&radius=20
You will see a field "D" in the resulting json which provides the exact distance (in miles) between the input lat/long and the campground.

http://api.campnear.me/GetAvailableFacilities

Retrieve all facilities that have availability. http://api.campnear.me/GetAvailableFacilities
