campnear.me API

What is this?

This API is a portal to the (Oregon and Washington) camping universe. Currently powering campnear.me this data set includes bits from RIDB API, Region 6 National Forest Service websites and reservation information from recreation.gov. Data from these sources has been consolidated, possibly deduped and cleaned, and augmented with information about drive times, public transit availability, and some special sauce features to provide the One Camping Database To Rule Them All™
